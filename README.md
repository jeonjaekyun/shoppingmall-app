# Shoppingmall-App

## 주제
Nodejs와 React를 통해 만든 Shoppingmall-App

## 개발 언어 및 IT Skill
- JavaScript
- nodejs, react, redux, express
- MongoDB, Mongoose

## 주요기능
- 로그인/회원가입
- 상품 등록/ 상품 상세 페이지
- 장바구니
- 상품평 등록(댓글)
